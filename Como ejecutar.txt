El archivo de python es el encargado de convertir en pdf los datos del formulario

Para ejecutarse primero hay que iniciar un entorno de "VENV"

Para esto (En Arch Linux):
    Python -m venv venv 
    Una vez la carpeta este generada
    source ./venv/bin/activate

En Windows:
    Instalar una versión moderna de Python 3 desde la Microsoft Store o desde el instalador oficial.
    
    Abrir la carpeta del proyecto en Visual Studio Code o en Símbolo del sistema / PowerShell.
    
    Crear el entorno virtual con:
        python -m venv venv

    Desde una PowerShell activarlo entrando al directorio donde se encuentre la carpeta de nombre "venv"
    .\venv\Scripts\Activate.ps1
    .\venv\Scripts\Activate.bat (En cmd)

El archivo necesita unas librerias. Recomendablemente instalar dentro del entorno de venv
    pip install flask
    pip install flask_cors
    pip install fpdf
    pip install fpdf2
